# E-commerce Customer Behavior Analysis

## Overview
Analyzed 1M+ e-commerce transactions using Python, SQL, and Power BI to uncover customer behavior, revenue trends, and product performance.

---

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (PostgreSQL)
- Power BI
- Jupyter Notebook

---

## Key Analysis Performed
- Customer buying patterns across demographics  
- Product performance based on ratings, orders, and revenue  
- Coupon usage vs spending behavior  
- Campaign performance analysis  
- Shipping method impact on purchases  
- Weekend vs weekday buying trends  

---

## Dashboard
The Power BI dashboard includes:
- Revenue by product category  
- Customer distribution  
- Campaign performance  
- Sales trends  

 (You can download it from `/dashboard/dashboard.pdf`)

---

## Key Insights
- High-rated products do not always generate the highest revenue  
- Coupon users often spend equal or more than average customers  
- Revenue contribution varies across age groups  
- Campaign sources impact both acquisition and revenue  
- Buying behavior differs between weekends and weekdays
